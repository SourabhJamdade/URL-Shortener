# URL Shortener

A simple URL Shortener application built with **Node.js**, **Express.js**, **MongoDB**, and **EJS**.

## Features

- User Authentication
- Generate Short URLs
- Redirect to Original URLs
- URL Analytics
- Protected Routes using Middleware

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS

## Installation

```bash
git clone https://github.com/sourabhjamdade/url-shortener.git

cd url-shortener

npm install
```

Create a `.env` file:

```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
```

Run the project:

```bash
npm start
```
